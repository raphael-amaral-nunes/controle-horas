# Controle de Horas

Sistema para controle de dias e horas trabalhadas.

## Como usar

### Requisitos
- Java 17 ou superior instalado

### Download
1. Baixe o arquivo `controle-horas.jar` da seção [Releases](https://github.com/seu-usuario/controle-horas/releases)
2. Salve em uma pasta de sua escolha
